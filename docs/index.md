# LG V30 Root 2026 - KernelSU Next

KernelSU Next root kernel for the **LG V30 joan / H930** running **LineageOS 22.2 Android 15**.

This project provides reproducible scripts and patches for building a KernelSU Next-enabled Linux 4.4.302 non-GKI kernel with manual hooks for the LG V30.

## Compatibility

- Device: LG V30
- Codename: `joan`
- Tested model: H930
- ROM: LineageOS 22.2
- Android: Android 15
- Kernel: Linux 4.4.302 non-GKI
- Root method: KernelSU Next legacy
